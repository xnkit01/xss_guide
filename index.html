Introduction

Cross-Site Scripting (XSS)

Cross-Site Scripting (XSS) is a widespread security vulnerability typically found in web applications. It allows attackers to inject malicious client-side scripts into web pages viewed by other users.

Unlike other attacks that target the server directly, XSS targets the application's users. When a victim visits an infected web page, their browser executes the malicious script, mistakenly believing it came from a trusted source.
